# Netflix Data Engineering Project

## Overview
Built an end-to-end data pipeline using PySpark and AWS S3.

## Steps
- Loaded raw Netflix dataset
- Cleaned and transformed data using PySpark
- Performed aggregations using Spark SQL
- Stored processed data in Parquet format
- Uploaded data to AWS S3

## Tech Stack
- Python
- PySpark
- AWS S3
- Parquet

## Key Learnings
- Handling large datasets using Spark
- Data cleaning and transformation
- Working with cloud storage (AWS S3)

## Output
- The processed data is stored in Parquet format using Apache Spark.
- Due to large size, output files are not uploaded to this repository.
